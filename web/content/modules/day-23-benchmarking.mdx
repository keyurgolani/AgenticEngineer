### 23.1 What is SWE-bench?

A dataset of **Real GitHub Issues** from popular python libraries (Django, Flask, scikit-learn).

- **Input**: The Issue Description + Codebase.
- **Goal**: Generate a Pull Request.
- **Verification**: Run the _new_ tests associated with that issue.
